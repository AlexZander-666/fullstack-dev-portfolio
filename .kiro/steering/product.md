# Product Overview

Personal website with a modern, Claude Code-inspired design. Full-stack application for showcasing skills, projects, and blog posts with admin content management capabilities.

## Core Features

- **Public Pages**: Hero section, skills showcase, project portfolio, blog with Markdown support
- **Content Management**: Admin dashboard for managing projects, blog posts, and contact messages
- **Authentication**: JWT-based auth for admin access
- **Contact Form**: Visitor contact submission with backend storage
- **SEO Optimized**: Server-side rendering, meta tags, and performance optimization (target: Lighthouse 90+)

## User Roles

- **Visitor**: Browse content, read blog posts, submit contact forms
- **Admin**: Full CRUD operations on projects, posts, and content management
